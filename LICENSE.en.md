# License and license key

## Concepts
- The RosGear web application is software hosted on a
remote server and accessible to users via the internet.
- RosGear web application components are modules, module extensions, plugins, and widgets
installed in the "/modules" directory of the web application, including program
code and supporting files.
- RosGear libraries are a set of PHP scripts located in the "/vendor/rosgear" directory of the web 
application.

## License key

A license key or product key (hereinafter referred to as the key) is a unique alphanumeric sequence that is specified 
during installation of the RosGear web application. If the web application is installed, the key is specified in the 
settings.

The key is used to install and update components and libraries of the web application. The key can be unique,
per user, for the entire web application, or for a server IP address. The key can have an expiration date and be renewed 
by the user.

The key can be paid or free. A paid key allows you to install commercial components or components with advanced 
functionality, as well as receive technical support and developer documentation.
